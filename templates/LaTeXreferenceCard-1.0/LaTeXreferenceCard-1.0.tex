\begin{center}
     \Large{\textbf{\LaTeX\ Reference Card Template}} \\
\end{center}

\LaTeX\ Reference Card Template. \newline
Use this template to create your own reference card. Actual reference cards
can be two pages or more.

\section{Downloading}
Download the \LaTeX\ Reference Card Template from
\href{http://AnthonyBradford.com/LaTeXreferenceCard/LaTeXreferenceCard-1.0.tar.gz}{LaTeXreferenceCard-1.0.tar.gz}.

\section{Extracting}
tar xfz LaTeXreferenceCard-1.0.tar.gz
\newline
cd LaTeXreferenceCard-1.0

\section{Building}
Type: make
\newline
See newly created PDF reference card.

\section{Spelling}
Type: make spelling
\newline
GNU aspell will be run in \TeX\ mode on the reference card.

\section{Images}
Keep images in the images/ directory. Makefile will automatically convert
PNG, JPG and GIF to EPS format.

\section{Distribution}
Type: make dist
\newline
Creates LaTeXreferenceCard-1.0.tar.gz distribution.
\newline
